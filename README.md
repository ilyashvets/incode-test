# Incode Group test task

<br>Framework: NestJS
<br>DB: MYSQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Migrations

```bash
# generate
$ npm run migration:generate

# run
$ npm run migration:run

# revert
$ npm run migration:revert
```
